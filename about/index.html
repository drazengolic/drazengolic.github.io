About me

I’ve been a professional software developer for over 11 years. I’ve started off as a junior web developer tinkering with websites, later doing some application development, but for the past several years I’ve been doing mostly server-side and database development.

My roots in software development go way back to the end of elementary school, when I’ve discovered QBASIC and Pascal. In my lifetime, I’ve put my hands on several programming languages and tools - I’ve built desktop apps, web sites, web apps, system integrations, and even some mobile games. I think of myself as a generalist and goal-driven, rather than being a specialist in narrow set of technologies. But that generalism has it’s limits though - I tend to stay away from insufferable world of modern UI development, but also from false promises of many other tech du jour whose number seems to have exploded in recent years.

This blog

This is where I’ll be posting (mostly technical) things for which I believe are worth sharing with the accidental visitors like yourself. My focus is on tools and techniques that make the simple solutions possible, rather than those that will fill out my CV but also make my hair gray before it’s due.
