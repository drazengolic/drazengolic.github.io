About me

I’ve been a professional software developer for over a decade. I’ve started off as a junior web developer tinkering with websites, later doing some application development, but for the past several years I’ve been doing mostly server-side and database development.

My roots in software development go way back to the end of elementary school, when I’ve discovered QBASIC and Pascal. In my lifetime, I’ve discovered and used several programming languages and tools - I’ve built desktop apps, web sites, web apps, system integrations, and even some mobile games. I think of myself as a generalist and goal-driven, rather than being a specialist in narrow set of technologies. But that generalism has it’s limits though - I’m not fond of technologies that are unnecessarily complicated (think modern web frameworks), so I try to avoid them whenever possible. But, I’m always on the quest of finding better, simpler ways to achieve things.

This blog

This is where I’ll be posting (mostly technical) things for which I believe are worth sharing with the accidental visitors like yourself, with more focus into doing rather than talking. When it comes to programming, I’m a non-conformist and sometimes may look opinionated about things, but opinions change with experience.
